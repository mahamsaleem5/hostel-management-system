# Hostel Management System (Python)

This Hostel Management System is a console-based Python project designed to manage hostel students, room allocation, and complaints efficiently. The system uses core data structures such as linked list, min heap, and queue to handle real-world hostel operations in an organized way.

## Features
- Add, search, delete, and display student records
- Automatic room allocation using Min Heap
- Room availability checking
- Complaint management using Queue (FIFO)
- Data saving and loading using file handling (pickle)
- Menu-driven and user-friendly interface

## Data Structures Used
- Linked List → Student records
- Min Heap → Room allocation
- Queue → Complaint management
- File Handling → Persistent data storage
